# CSE 210 - Programming with Classes

This repository contains C# projects developed as part of the CSE 210 Programming with Classes course. The projects are organized by weeks, demonstrating various object-oriented programming concepts and practices.

## Project Structure

sandbox/: A playground area for experimenting with C# concepts
week01/: Basic C# programming exercises
week02/: Projects including Journal and Resumes applications
week03/: Projects focusing on Fractions and Scripture Memorizer
week04/: Applications for YouTube Videos and Online Ordering systems
week05/: Homework tracker and Mindfulness applications
week06/: Projects exploring Shapes and Eternal Quest concepts
week07/: Exercise Tracking application


## Getting Started

1. Ensure you have .NET SDK installed
2. Clone this repository
3. Navigate to any project folder
4. Run: dotnet build (to compile)
5. Run: dotnet run (to execute the program)
